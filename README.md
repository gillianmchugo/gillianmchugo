
# [Gillian McHugo](https://gillianmchugo.github.io) :cow2::dna::computer:

Computational biology PhD researcher at University College Dublin
studying integrative genomics of infectious diseases in cattle, focusing
on trypanosomiasis in hybrid African cattle and bovine tuberculosis in
Irish cattle.

[gillianmchugo.github.io](https://gillianmchugo.github.io)
[orcid.org/0000-0001-6920-0041](https://orcid.org/0000-0001-6920-0041)
